Running Pace and Distance by Time

I wanted a way to visualise my running data taken from Strava. This interactive plot was my answer! The bubble is the distance of each run and the Y-axis is the pace of the run. The original version of this also plotted the runs onto a map, but I can't put that version on a static website! I also haven't set this up to download directly so it uses a static version of the data (its missing my most recent runs).
