Step 1: Setting up a web-server

When developing our D3 plots, the first step will be to open the data. Unfortunately, browsers generally can't load local files for security reasons, so to develop the plot locally, you generally need to create a web-server. This process is quite simple but requires either python, node.js or something similar to be installed. Instructions for how to use different web servers can be found here:

Loading Data

d3.csv( "MyData.csv" ).then( function(data) {
[...FUNCTION BODY...]
});

This opens the csv file and then passes the data from the file directly into a new function as a value called data. The function created below it is then run.
