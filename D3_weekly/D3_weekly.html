D3 WEEKLY

I'm trying to build up a portfolio and practice log of using D3.js to visualise data. The visuals will use data from a mix of sources, with the priority being making interesting visuals rather than showing important data. I will aim to pick somewhat interesting datasets though!

I hope to do 2-3 visuals a week but I honestly have no idea how long I'll be able to keep the momentum up...

Update:

Instead of making individual plots from random data sources, I have been working on creating a library of Reusable plots - this follows the structure defined by Mike Bostock to make an easy to reuse libray. The current state of this library is here.
The aim of this is to straight-forwardly create a data dashboard by using previously created plots, rather than making new ones from scratch each time. Of course, a certain amount of customisation would always be beneficial, but I am aiming to allow rapid generation of good looking dashbaords.
